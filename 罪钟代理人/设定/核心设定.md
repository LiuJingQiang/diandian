# 核心设定表

## 基本信息

- 书名：我在烬岚城替人承罪
- 题材/类型：轻小说 / 原生幻想 / 东方蒸汽 / 终末逆命
- 目标平台：男频轻小说向
- 目标读者：中国、日本、韩国等东亚读者；偏好近代都市压迫、底层逆命、制度漏洞爽点和危险女性搭档
- 背景定位：异世界东方幻想城邦。地名、官阶、机构和灾厄制度均为虚构，只保留东亚读者熟悉的家族、街坊、状纸、学舍、官司压迫等情绪入口。

## 一句话梗概

烬岚城鸣钟司代书季暮舟在未来案卷中看见檐灯巷将被当成疫源焚街。为了救母亲和整条巷子，他用代书吏的职权递上代理认罪状，替白铃认下“私鸣灾钟”之罪，成为能从罪名中夺取权柄的承罪代理人。

## “承罪代理人”是什么意思

烬岚城的罪钟不是普通钟，而是城廷、神社式钟楼和炉商会共同维护的审判机关。每当大灾发生，罪钟必须登记一个“承罪人”，城市才会停止扩大清算。

代理人不是随便替人顶罪的人，而是鸣钟司制度里的灰色职务：

1. 被告不能出庭、身份被抹除或案卷不完整时，代书吏可以提交“代理认罪状”。
2. 代理人签押后，罪名从原人身上转到代理人身上。
3. 罪名会带来刑罚和侵蚀，但也会释放对应的“钟权”。
4. 季暮舟原本只是写字盖章的小吏，第一次承罪后才发现代理认罪不是文书漏洞，而是罪钟最早的隐藏权柄。

## 主角设定

- 姓名：季暮舟
- 年龄：20
- 身份：烬岚城灰檐区鸣钟司代书吏，负责替穷人写状纸、抄案卷、盖边章
- 核心特质：会算账、会忍、嘴上冷淡但有底线
- 金手指：未来案卷残页 + 代理承罪权柄
- 弱点：母亲病重，妹妹在星灯学舍读书，自己只是临时小吏，随时会被顶包
- 核心动机：保住檐灯巷，查清是谁把整条巷子写成疫源

## 世界观骨架

- 城市：烬岚城，悬在黑雾海边缘的东方幻想蒸汽城邦，钟楼、灵轨车、煤烟、晶票庄、鸣钟司、铃社式钟楼并存
- 核心制度：灾事发生后，罪钟登记罪名和承罪人，城廷据此封街、抄家、焚屋或赦免
- 力量体系：罪名不是道德评价，而是被罪钟承认的灾厄契约。承罪越深，权柄越强，身体越接近钟鬼
- 社会结构：云阶区司族和炉商会决定谁有罪，灰檐区百姓承担罪名和灾后清算

## 核心冲突

- 主线矛盾：季暮舟要救檐灯巷，就必须不断替关键人物承罪，利用罪钟规则反过来咬住制定规则的人
- 终极阻碍：鸣钟司高层和云阶区炉商会用罪钟制造替罪羊，把每次灾难的成本转嫁给灰檐区
