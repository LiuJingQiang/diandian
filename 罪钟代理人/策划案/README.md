## 一、核心玩法框架

### 1.1 主循环（一句话）
> **读段落 → 遇岔路（付费/广告/符册锁）→ 消耗钟能做选择 → 承罪层数↑ / 信任↑ / 黑雾侵蚀↑ → 解锁钟权，推动檐灯巷存亡结局**

### 1.2 三层变现结构（在点点双层基础上叠加"侵蚀去污"）

```
第一层：即时消费              第二层：属性门槛            第三层：负资源清算
──────────────               ──────────────              ──────────────
抉择选项 (12 钟能)     ──┐    承罪门槛选项(需 承罪≥N)   黑雾侵蚀累积 ↑
灾罪章解锁 (17 钟能)   ──┼──► 权柄/信任/承罪 数值池 ◄─ 陆青萝药包 (¥课金去侵蚀)
广告替代 (免费)        ──┘    符册阁贴符 (+属性/信任)   侵蚀满 = 钟鬼 BE
```

**核心设计原则**：
1. **每岔路 ≥ 2 平行付费选项**（选 A "递代理认罪状" +承罪+权柄，选 B "沉默" +警觉降低）— 每一步都变现
2. **承罪机制自带戏剧张力**：每次替人承罪 → 得钟权（爽）+ 得侵蚀（毒），玩家必须课金去毒
3. **檐灯巷存亡 = 街坊 HP 条**：全局唯一，跌到 0 直接 BE，激发"救灾支付"动机

### 3.6 承罪 ↔ 侵蚀 联动公式（核心）

```
每次承罪选项:
  sin_layers += 1
  bell_authority += 10 (随承罪档位增)
  dark_mist += 5 ~ 20 (承罪越大侵蚀越深)

清算方式:
  - 陆青萝解毒药包: -20 侵蚀 / 瓶 (售 50 钟能 or ¥3 现金包)
  - 白铃陪伴选项: -10 侵蚀 (需 trust_bailing >= 100)
  - VIP 日衰减: -5/天
  - 睡眠/时间流逝: -2/天 (无 VIP)

死亡条件:
  dark_mist >= 100     -> 变钟鬼 BE
  alley_survival <= 0  -> 檐灯巷毁灭 BE
  alert_bellsi >= 100  -> 通缉 BE
```
