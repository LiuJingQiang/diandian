# 前端策划案（game-frontend 侧）

> **应用**：`game-frontend/` (React + Vite · 8788)
> **主题**：东方蒸汽 · 男频轻小说 · 灰檐区代书吏
> **配置源**：`罪钟代理人/策划案/配置/*.json` (与后端共享 · 后端权威)
> **原则**：**前端只渲染 + 收集意图**，所有涉及经济/变量/结局的判定发给后端

---

## 一、前端职责边界

### 🟢 前端**必须做**

| 模块 | 职责 |
|---|---|
| **章节阅读** | paragraph 流式渲染 · 打字机效果 · CG 播放 · 音效 |
| **选择 UI** | choice 平行选项按钮 · 显示成本 12 钟能 / 广告 icon / 门槛 icon |
| **变量面板** | 承罪层数 / 权柄 / 信任度 / 侵蚀 / 檐灯巷存亡 全变量可视化 |
| **HUD** | 顶部钟能余额 · 侵蚀条 · 章节进度 |
| **商店入口** | 4 大入口: 钟能 / 钟符 / 未来案卷 / 香火愿力 + 陆青萝药包（独立入口）|
| **引导 tour** | Ch4 首视觉章弹窗 · Ch5 首解毒弹窗 · Ch13 檐灯巷告急弹窗 |
| **主题化命名** | 所有语义 key → 中文题材化（energy→钟能, wardrobe→符册阁）|
| **本地 flag** | "看过 tour" · "视频引导已完成" · 语言设置 · 音效开关 |
| **错误处理** | 402/423/429 → 展示对应引导（充钟能/贴符/看广告）|
| **动画/特效** | 三息回钟时间倒流特效 · 侵蚀条黑雾滤镜 · 敲钟震动 |

### 🔴 前端**不能做**

- **不能自己扣钟能** → 一律 POST `/api/choice/select`
- **不能自己判定门槛** → 服务端返 GATE_FAIL 再展示
- **不能自己发抽卡结果** → 一律 POST `/api/gacha/pull`
- **不能自己解锁章节 / 送奖励** → 全服务端结算
- **不能改配置**（同一份配置读只读）

**关键交互**：
1. 打字机效果 60ms/字，点屏加速
2. 视觉章前置弹窗：`"下一节含 CG · 需 17 钟能解锁"` → 3 按钮：`解锁 / 看广告 / 稍后`

4. 门槛不达时禁用点击 → 展示 hint 文字 + "去符册阁" 快捷入口
5. 结算后**变量面板浮起动画**：+X 好感 / -Y 侵蚀 变化数字飞出

### 3.2 侵蚀 HUD（本作独有 · 关键变现钩子）

```

  30-60% → 屏幕边缘微黑雾滤镜（CSS filter: drop-shadow）
  60-80% → 全屏黑雾+咳嗽音效随机播放

  =100%  → 触发 be_corruption 动画 → 跳结局页
```

```typescript
// game-frontend/src/api/client.ts

export class BellApiClient {
  async getUserState(): Promise<UserState> { ... }

    Promise<{ new_state, effects_applied } | { code: 'GATE_FAIL', need, have }>
  async unlockTalisman(id) { ... }
  async wearTalisman(id, equip) { ... }
  async gachaPull(pool, count) { ... }
  async createOrder(good_id) { ... }
  async verifyOrder(order_id, sign) { ... }
  async claimSignIn() { ... }
  async claimReadMilestone(minutes) { ... }
  async claimAdReward(ad_token) { ... }
  async evaluateEnding(chapter) { ... }
}
```

**错误处理规范**：
- 402 INSUFFICIENT_ENERGY → 弹「钟能不足 · 立即充值 or 看广告」
- 423 GATE_FAIL → 弹「需 {need} · 当前 {have} · 去符册阁」
- 429 AD_QUOTA_EXCEEDED → 灰化广告按钮 + toast「今日广告已达上限」
- 409 STATE_VERSION_STALE → 强制拉最新 state 后重试
- 网络错误 → 无限重试队列（乐观 UI 保留玩家操作意图）

## 九、性能预算

- 首屏 TTI < 2s (静态 HTML + 关键 CSS inline)
- 章节切换 < 300ms (预加载下章 paragraph + CG)
- CG 图片总量 < 3MB / 章 (WebP · lazy load)
- 打字机流畅度 ≥ 60 FPS
- 支持离线阅读缓存（已解锁章节）

---

## 十、无障碍

- 按钮 hit area ≥ 44px
- 文字对比度 ≥ 4.5:1
- 支持键盘导航（Tab / Enter 选择）
- 支持屏幕阅读器 ARIA 标签
- 可关闭：打字机 / 音效 / 动画（settings 页面）
