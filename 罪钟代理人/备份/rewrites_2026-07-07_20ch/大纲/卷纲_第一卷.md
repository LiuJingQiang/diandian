# 卷纲：第一卷 檐灯巷不能烧

## 功能

铺开季暮舟的生活压力、鸣钟司规则、代理认罪状逻辑、白铃关系、韩照追捕和檐灯巷焚街危机。

## 情绪弧线

生活压迫 -> 规则解释 -> 反常识承罪 -> 权柄兑现 -> 街坊代入 -> 追捕智斗 -> 身世钩子。

## 章节安排

| 章节 | 章节定位 | 核心事件 | 情绪 |
|---|---|---|---|
| 第001章 | 高压开篇 | 季暮舟为救檐灯巷替白铃递代理认罪状 | 代入、震惊 |
| 第002章 | 爽点兑现 | 代理承罪成功，季暮舟获得三息回钟 | 暗爽、逃亡 |
| 第003章 | 代入推进 | 季暮舟回到檐灯巷，韩照逼他交人或封巷 | 街坊压力 |
| 第004章 | 规则深化 | 陆青萝解释承罪侵蚀，檐灯巷验街告示出现 | 代价、紧迫 |
| 第005章 | 调查推进 | 季暮舟和白铃潜入旧钟楼寻找黑雾账本 | 智斗、悬疑 |
| 第006章 | 阶段反转 | 季暮舟发现自己出生时已是预备承罪人 | 身世、长线 |

## 伏笔

- 季暮舟为什么二十年前就被写成预备承罪人。
- 白铃为何私鸣灾钟，她知道多少旧钟楼秘密。
- 檐灯巷隐匿疫源案的真正灾名是什么。
- 陆青萝为什么会成为第二个浮现的名字。
